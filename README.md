# Gold Price Prediction 💰

## 📌 Description

This project predicts gold prices using machine learning techniques.
It uses historical data to train a model and forecast future prices.

---

## ⚙️ Technologies Used

* Python 
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📊 Features

* Data preprocessing and cleaning
* Visualization of gold price trends
* Machine learning model for prediction
* Performance evaluation

---

## 📂 Dataset

The dataset contains historical gold price values used for training and testing the model.

---

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells step by step
3. View predictions and graphs

## 📈 Output

* Graphs showing gold price trends
* Predicted values using ML model
